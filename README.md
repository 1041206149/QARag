# RAG问答机器人项目## 项目简介基于检索增强生成(RAG)技术的智能问答系统，结合知识库和大语言模型提供准确的问答服务。## 项目结构```qa_rag_bot/├── data/                    # 数据目录├── src/                     # 源代码├── vector_store/            # 向量数据库├── config/                  # 配置文件├── tests/                   # 测试代码├── notebooks/               # 演示笔记本├── logs/                    # 日志文件└── main.py                  # 主程序入口```## 功能特性- ✅ 基于语义检索的知识库查询- ✅ 大语言模型智能回答生成- ✅ 混合检索策略(语义+关键词)- ✅ 可配置的检索和生成参数## 技术栈- 向量化：sentence-transformers / OpenAI Embedding- 向量数据库：FAISS / Chroma- LLM：OpenAI API / 本地模型- 框架：Python 3.8+## 快速开始待补充...## 配置说明待补充...## 使用方法待补充...## 开发计划- [ ] 数据加载与预处理- [ ] 向量化与索引构建- [ ] 检索模块实现- [ ] LLM集成- [ ] RAG流程整合- [ ] 用户界面开发
